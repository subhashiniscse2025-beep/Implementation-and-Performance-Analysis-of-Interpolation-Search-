# Implementation-and-Performance-Analysis-of-Interpolation-Search-
This project implements both searching algorithms, compares their execution time and number of comparisons on datasets of different sizes, and allows users to search for elements in a sorted array through console input.

## Features
Implements Binary Search and Interpolation Search algorithms,
Accepts user input for the sorted array and target element,
Searches and locates the target element efficiently,
Counts the number of comparisons performed by each algorithm,
Measures and compares execution time,
Performs performance analysis on datasets of various sizes,
Displays search results with index position and comparison count,
Developed using Python with a simple command-line interface.
